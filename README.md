# shiori_extension

この拡張機能は、旅行中にウェブサイトを閲覧しながら地点を登録し、最終的に「旅のしおり」を作成できるChrome拡張機能です。

Google Map部分はAWS上で静的サイトとしてホスティングされています。

## セットアップ方法

Chromeの拡張機能ページで以下の手順を実行し、`maps` フォルダを読み込んでください。

拡張機能 > 拡張機能を管理 > パッケージ化されていない拡張機能を読み込む
